# FrameBox

FrameBox 是一款面向 Windows 的本地电影库管理工具，用于整理分散在多个硬盘或目录中的本地视频文件。它支持选择电影源目录、自动读取视频文件并归入未分类区，随后由用户手动补充影片番号、标题、年份、主演、类型、封面、剧照、备注等信息。软件使用本地 SQLite 数据库存储资料，不依赖云端服务，也不会从软件中删除真实视频文件。

FrameBox 的目标是提供一个干净、可控、适合个人收藏管理的本地电影库体验：你可以按标题、主演或番号搜索影片，编辑完整影片信息，管理封面和剧照，并通过本地播放器直接播放对应视频文件。后续计划继续扩展自动刮削、更多筛选方式、数据备份与更完善的媒体信息管理能力。
